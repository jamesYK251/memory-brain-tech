# Do YOUKNOW?

A memory card matching game that tests your ability to recognize the software stack that YOUKNOW Technologies resells across Africa.

## The Game

Flip cards to find matching technology partner logos. Fewer moves + faster time = higher score!

**15 technology partners** across 7 categories:

| Category | Partners |
|----------|----------|
| Analytics & BI | Amplitude, DOMO |
| CRM | HubSpot |
| Customer Engagement | Braze, OneSignal |
| Social | Brandwatch, Hootsuite, Rival IQ, CisionOne |
| Research & Insights | Audiense |
| Search Intelligence | Trajaan |
| Mobile Attribution | AppsFlyer, Branch |
| Exclusive African Partners | GWI, Khoros |

**Difficulty levels:**
- Easy — 6 pairs (12 cards)
- Medium — 10 pairs (20 cards)
- Hard — 15 pairs (30 cards, all partners)

## Tech Stack

- **Next.js 15** (App Router) + TypeScript
- **Tailwind CSS** with YOUKNOW brand tokens
- **Framer Motion** for 3D card flip animations

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build
